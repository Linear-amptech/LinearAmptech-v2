# LinearAmptech v2

## Requirements

- Node.js `22.x`
- `pnpm` `11.0.8`

## Install

```bash
pnpm install
```

## Run locally

```bash
pnpm start
```

The app runs at `http://localhost:3000`.

## Build

```bash
pnpm build
```

## Test

```bash
pnpm test
```

## DigitalOcean App Platform

Use this project as a `Static Site`.

- Build command: `pnpm build`
- Output directory: `build`
- Run command: leave empty
